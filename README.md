# Herbify (Herbal Garden E-commerce Website)

## Abstract
Herbify is a web-based platform designed for people interested in health and herbal remedies. It provides a professional information station for herb knowledge, including identification, medicinal uses, and herbal vocabulary. Users can also exchange ideas using local/common language.

Herbify is also an e-commerce platform to buy and sell medicinal and herbal plants. The goal is to revive the traditional use of medicinal plants as home remedies and increase their availability. It encourages people to create herbal gardens in homesteads, school campuses, and other institutions.

## Tech Stack
- Frontend: HTML, CSS, Bootstrap, Django Templates
- Backend: Django, MySQL

## Modules

### Admin Module

- Approve Users
- View Plant Payments
- View Machines
- View Registration Details

### User Module
- View Profile
- View Classes
- Add Plant Booking
- View Booking
- Tools and Machinery
- View Plant Payments
- View Machinery Payments

### Farmer Module
- Add Plant
- View Profile
- View Booking Plant
- Book Tools & Machinery
- View Tracking
- View Machines
- View Booking History

  ## How to Run
1. Clone the repository: `git clone https://github.com/Abhiramiramesh/Herbify.git`
2. Navigate to project directory
3. Install requirements: `pip install -r requirements.txt`
4. Run Django server: `python manage.py runserver`
5. Open browser at `http://127.0.0.1:8000/`
