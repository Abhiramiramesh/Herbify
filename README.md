# Herbify

## Description
Herbify is a Django-based e-commerce website focused on selling herbs and organic products online. It allows users to browse, select, and order products easily.

## Features
- User registration and login
- Product listing with categories
- Add to cart and checkout
- Admin panel for product and order management
- Simple and responsive UI

## Tech Stack
- Python, Django
- HTML, CSS
- SQLite

## How to Run
1. Clone the repository: `git clone https://github.com/Abhiramiramesh/Herbify.git`
2. Navigate to project directory
3. Install requirements: `pip install -r requirements.txt`
4. Run Django server: `python manage.py runserver`
5. Open browser at `http://127.0.0.1:8000/`
